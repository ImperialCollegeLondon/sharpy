class sharpy.aero.models.aerogrid.Aerogrid[source]¶
static compute_gamma_dot(dt, tstep, previous_tsteps)[source]¶

Computes the temporal derivative of gamma using finite differences. It will use a first order approximation for the first evaluation (when len(previous_tsteps) == 1), and then second order ones. :param dt: delta time for the finite differences :param tstep: tstep at time n (current) :param previous_tsteps: previous tstep structure in order: [n-…, n-2, n-1] :return:
